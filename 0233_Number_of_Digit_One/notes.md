# 233. Number of Digit One — 题解笔记

## 题意

给定整数 `n`，统计从 `0` 到 `n` 的所有整数中，数字 `1` 总共出现了多少次。核心难点在于不能逐个枚举到 `n`，而要利用数位规律按位统计。

## 核心算法

### 算法一：暴力枚举
**思路**：从 `1` 枚举到 `n`，对每个数不断取模和整除，统计其中有多少个数位等于 `1`。实现最直接，但完全没有利用数位分布的规律。`n` 到 `10^9` 时会明显超时。  
**时间**：O(n log n)　**空间**：O(1)

### 算法二：数位 DP
**思路**：把 `n` 转成字符串，从高位到低位递归枚举当前位填什么数字，并记录“前面已经放了多少个 `1`”或者“后续子树能贡献多少个 `1`”。通过 `isLimit` 控制当前位是否仍受 `n` 约束，并对不受限状态做记忆化。  
**时间**：O((log n)^2)　**空间**：O((log n)^2)

### 算法三：按位贡献统计
**思路**：分别计算个位、十位、百位等每一位上数字 `1` 出现了多少次，再把所有位的贡献相加。对某一位 `digit=10^k`，只需看该位左边的高位、该位本身 `cur` 和右边的低位：`cur=0` 时贡献 `high*digit`，`cur=1` 时贡献 `high*digit+low+1`，`cur>1` 时贡献 `(high+1)*digit`。本质上是按完整循环段和残缺段来计数。  
**时间**：O(log n)　**空间**：O(1)
