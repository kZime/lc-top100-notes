# 122. Best Time to Buy and Sell Stock II — 题解笔记

## 题意

给定每日股价数组，允许无限次买卖（同一天可先卖再买），但任意时刻最多持有一股，求最大总利润。

## 核心算法

### 算法一：O(n²) DP（枚举最后买入日）

**思路**：`dp[i]` 表示在第 `i` 天卖出时能获得的最大总利润。对每个 `i`，枚举所有 `j < i`，状态转移为 `dp[i] = max(dp[j] + max(0, price[i] - price[j]))`。含义是：在已完成若干笔交易（最后一笔在第 j 天卖出）的基础上，若第 j 天买入第 i 天卖出有利可图则加上这笔收益。答案为 `dp[n-1]`。

**时间**：O(n²)　**空间**：O(n)

### 算法二：O(n) DP（持仓状态机）

**思路**：用两个状态建模——`dp[i][0]` 第 i 天结束时手中无股的最大利润，`dp[i][1]` 手中有股的最大利润。转移：`dp[i][0] = max(dp[i-1][0], dp[i-1][1]+prices[i])`（保持空仓 or 今日卖出），`dp[i][1] = max(dp[i-1][1], dp[i-1][0]-prices[i])`（保持持仓 or 今日买入）。因为只依赖前一天，可压缩到 O(1) 空间。

**时间**：O(n)　**空间**：O(1)

### 算法三：贪心（最优）

**思路**：连续上涨段的总收益等于每相邻两天正差价之和（`p3-p1 = (p2-p1)+(p3-p2)`）。因此只需遍历一遍，将所有 `prices[i]-prices[i-1] > 0` 的部分累加即得最大利润。无需显式追踪买入/卖出点。

**时间**：O(n)　**空间**：O(1)

## 我的解法

### [solution.cpp](solution.cpp) — 算法一：O(n²) DP

**判断依据**：代码维护 `dp[i]`，外层枚举当前天 `i`，内层枚举所有买入日 `j`，以 `dp[j] + max(0, price[i]-price[j])` 更新，与算法一完全吻合。

**可改进点**：
1. **时间复杂度**：当前 O(n²)，算法三贪心仅需一次遍历即可达到 O(n)，代码量更少。
2. **边界处理**：若 `n == 1` 时 `dp[0]` 未被初始化为 0，实际上 `vector<int> dp(n)` 已默认初始化为 0，但若 `n == 0` 会出现越界（题目约束 n≥1，可暂不处理）。

### [solution1.cpp](solution1.cpp) — 算法二：O(n) 持仓状态机 DP

**判断依据**：维护 `dp[i][0]`（空仓）和 `dp[i][1]`（持仓）两个状态，转移方程 `dp[i][0] = max(dp[i-1][0], dp[i-1][1]+price[i])`、`dp[i][1] = max(dp[i-1][0]-price[i], dp[i-1][1])`，与算法二完全一致。

**可改进点**：用两个滚动变量替换 `vector<array<int,2>>` 即可将空间压缩到 O(1)，逻辑不变。

### [solution2.cpp](solution2.cpp) — 算法三：贪心（最优）

**判断依据**：一次遍历，`ans += max(0, price[i]-price[i-1])`，即累加所有相邻正差价，与贪心核心完全吻合。

**可改进点**：无。O(n) 时间、O(1) 空间，已是最优解。
