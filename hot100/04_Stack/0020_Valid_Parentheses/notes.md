# 20. Valid Parentheses — 题解笔记

## 题意

给定只含 `()[]{}` 的字符串，判断括号是否全部合法闭合：左括号必须由相同类型的右括号按正确顺序闭合。

## 核心算法

### 算法一：暴力字符串替换

**思路**：反复用空串替换所有合法配对子串 `()`、`[]`、`{}`，直到字符串不再变化。若最终为空则合法。每轮扫描一遍并消除所有内层配对，需多轮直到无配对可消。
**时间**：O(n²)　**空间**：O(n)

### 算法二：辅助栈（最优）

**思路**：遍历字符串，遇到左括号入栈；遇到右括号时检查栈顶是否为对应的左括号，是则弹栈，否则直接返回 `false`。遍历结束后栈为空则合法。利用栈的后进先出特性天然契合括号嵌套结构。可额外加奇数长度的早退优化。
**时间**：O(n)　**空间**：O(n)

## 我的解法

### [solution.cpp](solution.cpp) — 算法二：辅助栈

**判断依据**：用 `stack<char>` 存左括号，遍历时左括号入栈，右括号时用下标位置匹配（`"([{"` 中位置 == `")]}"` 中位置）确认配对，不匹配立即 `return false`，最终返回 `stk.empty()`。逻辑与最优辅助栈完全一致。

**可改进点**：用 `unordered_map<char,char>` 建右括号→左括号映射，替代两个字符串的下标比较，可读性更高；另可在开头加 `if (s.size() % 2) return false;` 提前剪枝奇数长度字符串。两点均不影响渐进复杂度。
