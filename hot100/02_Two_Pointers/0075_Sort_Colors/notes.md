# 75. Sort Colors — 题解笔记

## 题意

给定只含 0、1、2 的数组，原地排序使三种值各自相邻（顺序为 0→1→2）。不能用库排序，进阶要求一趟扫描 + O(1) 额外空间。

## 核心算法

### 算法一：计数排序（两遍扫描）

**思路**：先统计 0、1、2 各自的数量，再按数量重写整个数组。两次遍历，实现简单直接。

**时间**：O(n)　**空间**：O(1)

### 算法二：单指针（两遍交换）

**思路**：第一遍将所有 0 交换到数组头部，第二遍将所有 1 交换到 0 之后。用一个指针 `ptr` 标记已排好的边界，每次找到目标值就与 `ptr` 位置交换并后移。两遍 O(n)，空间 O(1)。

**时间**：O(n)　**空间**：O(1)

### 算法三：荷兰国旗（三指针，最优）

**思路**：三指针 `lt`、`i`、`rt` 将数组划分为三段：`[0, lt)` 全为 0，`[lt, i)` 全为 1，`(rt, n-1]` 全为 2，`[i, rt]` 为待处理区间。遍历时：`nums[i]==0` 则与 `lt` 交换并同时推进 `i` 和 `lt`；`nums[i]==2` 则与 `rt` 交换并收缩 `rt`（`i` 不推进，因换来的元素未检查）；`nums[i]==1` 只推进 `i`。一趟扫描完成排序。

**时间**：O(n)　**空间**：O(1)

## 我的解法

### [solution.cpp](solution.cpp) — 算法三：荷兰国旗（三指针）

**判断依据**：使用 `lt`、`i`、`rt` 三指针，`i` 在 `[i, rt]` 区间内扫描，对 0 向左交换并双指针推进，对 2 向右交换只收缩 `rt`，对 1 仅推进 `i`——与荷兰国旗算法完全吻合，一趟 O(n)，O(1) 空间，已是最优解。
