# 239. 滑动窗口最大值 — 题解笔记

## 题意

给定数组 nums 和窗口大小 k，返回每个大小为 k 的滑动窗口的最大值组成的数组。核心挑战是当最大值被移出窗口时如何高效找到新的最大值。

## 核心算法

### 算法一：优先队列（大根堆）
**思路**：维护存 `{val, idx}` 的大根堆。r 右移时入堆；每次取堆顶时检查其 idx 是否在窗口 `[i-k+1, i]` 内，若不在则弹出，直到堆顶在窗口内为止，作为当前最大值。
**时间**：O(n log n)　**空间**：O(n)

### 算法二：单调递减双端队列（最优）
**思路**：维护一个存 `{idx, val}` 的双端队列，队列保持严格递减序（队头是当前窗口最大值）。r 右移时：先从队尾弹出所有 val ≤ nums[r] 的元素（它们永远不会成为后续窗口的最大值），再将 nums[r] 入队尾；然后从队头弹出 idx ≤ i-k 的过期元素。当 i ≥ k-1 时，队头即为当前窗口最大值。每个元素至多入队/出队一次。
**时间**：O(n)　**空间**：O(k)

## 我的解法

### [solution.cpp](solution.cpp) — 算法二：单调递减双端队列
**判断依据**：使用 `deque<pair<int,int>>` 存 `{idx, val}`，从队尾弹出 ≤ nums[i] 的元素保持递减，从队头弹出超出窗口的过期索引，i ≥ k-1 时取队头值，与单调队列算法完全一致。
复杂度 O(n) / O(k)，已达最优。
